4.5 🤖 Objetos Inteligentes: Integrando Machine Learning#

Aquí es donde la Ingeniería de Software potencia la Ciencia de Datos.

En lugar de tener un notebook desordenado llamado:

analisis_final_v3_real_definitivo.ipynb

…vamos a encapsular la inteligencia dentro de nuestros objetos de dominio.

🎯 Objetivo#

Queremos que nuestro objeto Instrumento pueda responder:

“Basado en mi historia… ¿cuál será mi precio mañana?”

🧭 Arquitectura del Instrumento Inteligente#

        flowchart LR
    A[Instrumento] --> B[MarketDataProvider]
    A --> C[Modelo ML]
    B --> D[Yahoo Finance]
    C --> E[Predicciones]

⚙️ Instalación#

poetry add scikit-learn yfinance

🧩 Implementación del Instrumento Predictivo#

import numpy as np
import pandas as pd
from sklearn.linear_model import LinearRegression
import yfinance as yf


class YahooFinanceClient:

    def obtener_historia(self, ticker: str) -> pd.DataFrame:
        t = yf.Ticker(ticker)
        return t.history(period="1y")


class Instrumento:

    def __init__(self, ticker: str, data_provider):
        self.ticker = ticker
        self.provider = data_provider
        self._history = pd.DataFrame()
        self._modelo = LinearRegression()
        self._entrenado = False

    def entrenar_modelo(self):
        """Entrena una regresión lineal sobre precios de cierre"""

        if self._history.empty:
            self._history = self.provider.obtener_historia(self.ticker)

        df = self._history.reset_index()

        X = df.index.values.reshape(-1, 1)
        y = df["Close"].values

        self._modelo.fit(X, y)
        self._entrenado = True

        print(f"🤖 Modelo entrenado para {self.ticker}")

    def predecir_precio(self, dias_futuros: int = 1) -> float:

        if not self._entrenado:
            self.entrenar_modelo()

        ultimo_indice = len(self._history)
        dia_objetivo = ultimo_indice + dias_futuros

        pred = self._modelo.predict([[dia_objetivo]])

        return float(pred[0])

🧪 Ejemplo de uso#

provider = YahooFinanceClient()

google = Instrumento("GOOGL", provider)

print("Precio estimado mañana:",
      google.predecir_precio(1))

print("Precio estimado en 30 días:",
      google.predecir_precio(30))

🤖 Modelo entrenado para GOOGL
Precio estimado mañana: 325.544790493636

📈 Visualización de Serie de Tiempo + Tendencia#

import matplotlib.pyplot as plt

hist = google._history.copy()

hist["t"] = range(len(hist))

modelo = google._modelo
hist["trend"] = modelo.predict(hist["t"].values.reshape(-1, 1))

plt.figure(figsize=(10, 5))
plt.plot(hist.index, hist["Close"])
plt.plot(hist.index, hist["trend"])
plt.title("Precio vs Tendencia")
plt.show()

🚀 ¿Qué logramos?#

Convertimos un objeto pasivo en un agente inteligente:

  • Antes: almacenaba precios

  • Ahora: proyecta tendencias

  • Integra datos externos

  • Ejecuta ML interno

Esto es Domain‑Driven Data Science.

🧪 Taller Final: El Oráculo Financiero#

🎯 Objetivo#

Construir una aplicación de consola que:

  • Consulte datos reales

  • Entrene modelos

  • Prediga tendencias

  • Recomiende acciones

🧭 Arquitectura del sistema#

        flowchart LR
    A[CLI App] --> B[Instrumento]
    B --> C[MarketDataProvider]
    B --> D[ML Model]
    C --> E[Yahoo API]

📋 Requerimientos#

1️⃣ Data Layer#

  • Implementar MarketDataProvider

  • Completar YahooFinanceClient

  • (Bonus) MockDataProvider desde CSV

2️⃣ Domain Layer#

Modificar Instrumento:

  • Inyección de dependencias

  • entrenar_modelo()

  • predecir_tendencia()

Modificar Posicion:

  • alerta_riesgo = True si pérdida > 10%

3️⃣ Aplicación CLI#

El script debe:

  1. Pedir ticker

  2. Consultar precio actual

  3. Predecir tendencia

  4. Preguntar compra

  5. Guardar JSON
